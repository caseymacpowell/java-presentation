Java

Not to be confused with Javascript

Created by: James Gosling
First Appeared: May 23, 1995
In California at Sun Microsystems

Purpose

  • Designed with the idea "write once, run anywhere" (WORA) in mind
  • Goal: portability, being able to have it run similarly on different harware and OS
  • Derives much of its syntax from C and C++
  • Used to create applications for your computer
  • Fun Fact: One of the most popular programming languages in use today

Examples

class ForLoopExample {
public static void main(String args[]){
for(int i=10; i>1; i--){
System.out.println("The value of i is: "+i);
}
}
}
